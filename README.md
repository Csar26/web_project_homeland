🌐 Responsive Web Project – BEM Architecture
📌 Project Overview
This project is a fully responsive website designed for three screen resolutions:
✅ 1280px (Desktop)
✅ 768px (Tablet)
✅ 320px (Mobile)

It follows the BEM (Block-Element-Modifier) methodology to ensure a well-structured, readable, and maintainable codebase. The project also implements CSS transformations, rotations, and various styling techniques through external stylesheets.

🎨 Key Features
BEM Architecture – Clean and modular CSS structure
Responsive Design – Optimized for different screen resolutions
CSS Transform & Rotate – Enhancing UI interactions
Flexbox Layout – Efficiently aligning and organizing content
🛠️ Technologies Used
HTML5 – Structuring the web content
CSS3 – Styling with a modular and scalable approach
Flexbox – Creating flexible and adaptive layouts
